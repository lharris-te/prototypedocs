# Dependency tree for ThousandEyes Enterprise Agent software

This document contains links to the dependency tree for the ThousandEyes [Enterprise Agent](https://success.thousandeyes.com/PublicArticlePage?articleIdParam=kA0E0000000CmnbKAC_What-is-an-Enterprise-Agent). We've exported the recursively-generated dependency trees of both the ThousandEyes Agent \(`te-agent`\) and the ThousandEyes [BrowserBot](https://success.thousandeyes.com/PublicArticlePage?articleIdParam=kA0E0000000CmnsKAC_What-is-the-BrowserBot) \(`te-browserbot`\) packages for each [supported operating system](https://success.thousandeyes.com/PublicArticlePage?articleIdParam=kA0E0000000CmnoKAC_Supported-Enterprise-Agent-operating-systems). The URLs leading to the files containing dependency trees can be found in the table below:

| Operating System | Architecture | te-agent | te-browserbot |
| :--- | :--- | :--- | :--- |
| Ubuntu Linux 16.04 | x86\_64 | [Latest](https://apt.thousandeyes.com/dists/xenial/te-agent-dependencies.txt) or [all versions](https://apt.thousandeyes.com/dists/xenial/) | [Latest](https://apt.thousandeyes.com/dists/xenial/te-browserbot-dependencies.txt) or [all versions](https://apt.thousandeyes.com/dists/xenial/) |
| Ubuntu Linux 18.04 | x86\_64 | [Latest](https://apt.thousandeyes.com/dists/bionic/te-agent-dependencies.txt) or [all versions](https://apt.thousandeyes.com/dists/bionic/) | [Latest](https://apt.thousandeyes.com/dists/bionic/te-browserbot-dependencies.txt) or [all versions](https://apt.thousandeyes.com/dists/bionic/) |
| CentOS 7 | x86\_64 | [Latest](https://yum.thousandeyes.com/CentOS/7/te-agent-dependencies.txt) or [all versions](https://yum.thousandeyes.com/CentOS/7/) | [Latest](https://yum.thousandeyes.com/CentOS/7/te-browserbot-dependencies.txt) or [all versions](https://yum.thousandeyes.com/CentOS/7/) |
| Red Hat Enterprise Linux 7 | x86\_64 | [Latest](https://yum.thousandeyes.com/RHEL/7/te-agent-dependencies.txt) or [all versions](https://yum.thousandeyes.com/RHEL/7/) | [Latest](https://yum.thousandeyes.com/RHEL/7/te-browserbot-dependencies.txt) or [all versions](https://yum.thousandeyes.com/RHEL/7/) |

For the CentOS 7, Red Hat Enterprise Linux 7, Ubuntu 16.04 and Ubuntu 18.04 operating systems, only x86\_64-based architectures are supported.
